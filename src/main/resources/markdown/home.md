# 接口文档首页

欢迎使用 SpringBoot 应用模板的 API 文档（Knife4j）。

- 本地开发环境：`http://localhost:8081`
- OpenAPI JSON：`/v3/api-docs`
- Knife4j UI：`/doc.html`

文档分组：

1. 用户模块（`/api/user/**`）
2. 认证模块（`/api/auth/**`）

安全说明：

- 需要在右上角 Authorize 中输入 Bearer Token（`Authorization: Bearer <token>`）
- 登录接口：`POST /api/auth/login`
